## 📖 진행 방식
1. 논문을 읽고 **핵심 아이디어를 리뷰**합니다.
2. 공식 구현 코드나 레퍼런스를 참고해 **핵심 모듈부터 부분 구현**을 진행합니다.
3. 익숙해지면 전체 모델을 재현하거나, 작은 데이터셋으로 실험합니다.
